# MaterialFlexLayout

El proyecto se genero con Angular 11

## Sobre el Proyecto

Es la estructura de un proyecto utilizando Angular Material y FlexLayout, con navbar, sidebar y contenedor principal de rutas.

## Ruteo

Incorpora ruteo a 3 modulos y sus respectivas paginas en forma de ejemplo a una clinica. Utiliza carga perezosa.

## API

No utiliza API ni servidor de datos. Todos los datos cargados son de fantasia y estan "Hardcodeados" dentro de la app.

## Autenticación

No tiene pantalla de login ni autenticacion de usuario.
